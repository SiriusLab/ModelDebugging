Abstract Syntax Project

What is an AS Project ?

An AS project is used to design an Abstract Syntax of a DSML (Domain Specific Modeling Language).

Definition of Abstract Syntax :

The Abstract Syntax of a DSML express , of structural way, the set of concepts and their relations.

Design a new Abstract Syntax :

GEMOC Studio provides tools to design an Abstract Syntax

  • An EMF Editor

What about Abstract Syntax and GEMOC project ?

The new Abstract Syntax designed or the existing one selected will be integrated into the GEMOC project in progress
