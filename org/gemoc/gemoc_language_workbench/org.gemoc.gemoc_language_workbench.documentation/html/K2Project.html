How to parameter a K2 project ?

It exists 2 features :

  • Create a new DSA project
  • Select an existing DSA project

How to create a new DSA Project ?

  1. Open the file "project.gemoc_language_conf"

    2. It shall refresh to point on the good gemoc language project

  2. Right click on this file, select Gemoc Language
  3. A wizard is opened itself, check create New DSA and click on button "Next"
  4. This wizard is opened now

  5. Fill the blanks
    • With an ecore file already featured in the gemoc language project

      • it is automatically featured

    • If no ecore file is featured yet in the gemoc language project

      • it shall be selected

  6. For more information on this wizard using

    7. more information

How to select an existing DSA Project ?

  1. Open the file "project.gemoc_language_conf"

    2. It shall refresh to point on the good gemoc language project

  2. Right click on this file, select Gemoc Language
  3. A wizard is opened itself, check select existing DSA and click on button "Next"
  4. This wizard is opened now

  5. Choose the project among those proposed
  6. Click on "OK" button to finish
